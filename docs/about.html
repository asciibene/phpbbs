About phpbbs

Welcome to a new kind of bulletin board(forum). I am developing phpBBS during my free time (I have alot of it unfortunately) solely with the use of a bluetooth keyboard along with an Android smartphone; i.e. no computer.
All that using rudimentary tools being pushed to their limit on constrained hardware. It is designed with the most minimalistic of attitudes and will stay that way.
I am confident that this website is hackable, also probably unoptimized. The code used to run this website is only a few weeks months old.Nevertheless, it will keep evolving regularly for as long as I can maintain it; adding new features as time passes.
